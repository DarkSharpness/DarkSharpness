### chayso 的字符串

#### 出题人 : DarkSharpness

#### 题目描述

chayso 先生面前有一串字符。出于好奇，他想要知道这一串字符里面有多少个大写字母，有多少个小写字母。请你写一个程序，帮他解决这个问题。

特别地，请不要使用字符数组。推荐用 cin/cout 读入字符，可以避免一些不必要的问题。

#### 输入描述

第一行一个正整数 n ，表示字符串的长度。

第二行一个长度为 n 的字符串。

保证 n 不超过 10 000 000 

#### 输出描述

第一行一个整数，表示小写字母有多少个

第二行一个整数，表示大写字母有多少个

#### 输入样例

```
5
ABcd0
```

#### 输出样例

```
2
2
```
