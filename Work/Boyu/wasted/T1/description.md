### 简单排序

#### 题目描述

对数组从小到大排序可以按照如下方式进行 ：首先找出数组中最小的数，将其和第一个数交换。忽略最小的数，找出剩下数组中的第二大的数，将其和第二个数交换。

以此类推，重复以上过程，即可将数组从小到大排序。

当然，你可能已经会一些更加高效的排序算法，没有关系，只要能完成数组从小到大排序的任务即可。

#### 输入描述

第一行一个正整数 n，表述数组的长度。

后面一行 n 个整数，表示数组的 n 个元素。

保证所有整数的绝对值不超过 100。

#### 输出描述

一行 n 个整数，将数组的元素从小到大地输出，中间用空格隔开。

#### 输入样例
```
5
1 4 1 3 2 
```
#### 输出样例
```
1 1 2 3 4
```
