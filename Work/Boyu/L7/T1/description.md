### Dark 教授的水题 3.0

#### 出题人 : DarkSharpness

#### 题目描述

众所周知，Dark 教授是一个 ASCII 画爱好者，从她之前出的诸多毒瘤题便可看出。为了降低难度，她决定让你输出单幅 ASCII 画，并且全部由字符 * 构成。

特别地，为了防止题目太简单，她规定你不能用 while 来实现循环，而必须用 for 来实现，即你的代码中不能出现 while 。

#### 输入描述

两个正整数整数 n,m 表示 ASCII 画的高度和宽度，保证不超过 100

#### 输出描述

指定的 ASCII 画

#### 输入样例

```
2 3
```

#### 输出样例

```
***
***
```
