### 讨厌的数字

#### 题目描述

请你统计在十进制小于 $10^n$ 的正整数中，数位中出现 $64$ 的数字的数量。

例如 $604$ 里面没有出现 $64$ ，但 $645$ 在数位中出现了 $64$ 。

由于答案可能很大，所以请你输出答案对 $1e9+7$ 取模的结果。

#### 输入描述

一个正整数 $n$ 。保证 $n$ 不超过 $1e5$ 。

#### 输出描述

一个正整数，表示在十进制小于 $10^n$ 的正整数中，数位中出现 $64$ 的数字的数量对 $1e9+7$ 取模的结果。

#### 输入样例

```
3
```

#### 输出样例

```
20
```
