### 关灯

#### 题目描述

现在，你有 $n\times n$ 个灯泡，每次按下一个按钮，你可以使得某一行或某一列的灯泡状态反转 (亮变暗，暗变亮)，求最少按下多少次按钮，可以使得所有的灯泡都变暗。

#### 输入描述

第一行一个正整数 $n$，表示行列数。

接下来 $n$ 行，每行 $n$ 个 $0$ 或 $1$ ，表示初始时灯泡的状态。 $0$ 表示暗， $1$ 表示亮。

保证 $n$ 不超过 $4$ .

#### 输出描述

一个正整数，表示最少按下按钮的次数。如果无解，输出 $-1$ 。

#### 输入样例

```
3
0 1 0
1 0 1
1 0 1
```

#### 输出样例

```
3
```
