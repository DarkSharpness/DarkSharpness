### 慧音老师的秋游

#### 题目描述

上白泽慧音是居住在人类村落的大教师。

一天，她带领学生们去迷途竹林秋游。很自觉的，学生们站成了一排，编号为 $1 \dots n$ 。然而由于人数实在是太多了，难以照顾到队尾的学生，因此她决定把学生们分为 $k$ 排，每排至少 $1$ 个人。

为了尽可能不打乱原有的队列，她不打算改变原有相对顺序，即同一排内的编号依然是连续递增的。

为了保证她能尽可能地看到每个学生，防止他们被妖怪抓住吃掉，她希望所有学生被遮挡的次数的总和至少。一个学生被遮挡次数定义为，他前面 (编号更小且位于同一排) 比他高的学生的数量。

#### 输入格式

第一行两个正整数 $n, k$ ，表示学生数量和分成的排数。

第二行 $n$ 个正整数，表示每个学生的身高 $h_i$ 。

保证 $1 \le k \le n \le 500$ 且 $1 \le h_i \le 10^9$ 。特别地， $k \lt 20$

#### 输出格式

一个正整数，表示最小的被遮挡次数总和。

#### 样例输入

```
8 3
20 50 30 60 40 100 5 1
```

#### 样例输出

```
2
```

#### 样例解释

一种最小遮挡方案如下:

- 第一排: $[20,50,30,60]$ 其中编号为 $3$ 的学生被遮挡 $1$ 次
- 第二排: $[40,100]$ 没有遮挡
- 第三排: $[5,1]$  编号 $8$ 的学生被遮挡 $1$ 次

总共被遮挡 $1 + 0 + 1 = 2$ 次。
