### 子树大小

#### 题目描述

给定一棵 $n$ 个节点的树，节点编号从 $1$ 到 $n$ 。

定义节点 $i$ 的子树大小为以节点 $i$ 为根的子树(包括 $i$ 本身)中的节点数量。

请你求出每个节点的子树大小。

#### 输入格式

第一行一个整数 $n$ 。

后面一行 $n$ 个整数，第 $i$ 个整数表示节点 $i$ 的父节点编号，根节点的父节点编号为 $0$ 。

#### 输出格式

一行 $n$ 个整数，表示节点 $1$ 到节点 $n$ 的子树大小。

#### 样例输入

```
7
2 0 2 1 2 3 4
```

#### 样例输出

```
3 7 2 2 1 1 1
```
